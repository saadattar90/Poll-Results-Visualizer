# 📊 Poll Results Visualizer

## 🚀 Overview
The Poll Results Visualizer is a Data Science project that analyzes survey/poll data and converts it into meaningful visual insights using Python.

This project helps organizations, researchers, and analysts understand public opinion, customer feedback, and survey trends through interactive and easy-to-understand charts.

---

## 🎯 Problem Statement
Poll and survey data is often raw, unstructured, and difficult to interpret manually.  

This project solves that problem by:
- Cleaning raw poll data
- Analyzing responses
- Visualizing trends using charts

---

## 💡 Solution
This system:
- Loads poll data from CSV
- Cleans and processes responses
- Generates visual insights like:
  - Bar Charts
  - Histograms
  - Line Charts
  - Word Clouds

---

## 🛠️ Tech Stack
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- WordCloud
